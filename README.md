# PlateRandomiser
Graphical interface for randomizing samples across plates (96, 384 wells) equally according to sample fingerprints.

Requirements:\
Developed under Python 3.8.12\
Packages used for implementation and testing:\
pyqt 5.9.2\
numpy 1.21.5\
pandas 1.4.1\
scikit-learn 1.0.2\
matplotlib 3.5.1
